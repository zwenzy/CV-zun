1. # Deskripsi aplikasi
    Aplikasi ini adalah Web Portfolio / CV Digital modern berbasis React JS yang dirancang untuk menampilkan profil mahasiswa secara profesional, interaktif, dan mudah dikelola. Aplikasi ini menggunakan konsep Data-Driven, di mana seluruh konten (Profil, Skill, Pengalaman) tidak ditulis langsung di dalam kode program (Hardcode), melainkan diambil dari file data terpisah (`JSON`). Hal ini memudahkan pengguna untuk memperbarui CV.

    # Teknologi Utama:
        React JS + Vite: Untuk performa website yang sangat cepat (Lightning Fast).
        CSS Variables: Styling terpusat di satu file untuk kemudahan ganti tema (Theming).
        Lucide React: Koleksi ikon modern dan ringan.
        JSON: Sebagai database sederhana untuk menyimpan data diri.

    # Fitur Utama:
    - Dynamic Profile:** Header profil lengkap dengan foto, bio, dan statistik.
    - Skill Bars:** Visualisasi tingkat keahlian (Advanced/Intermediate) dengan progress bar otomatis.
    - Experience Timeline:** Linimasa pengalaman kerja/organisasi yang otomatis mengurutkan posisi "Current" (sedang berjalan) ke paling atas.
    - Multi-Theme Support: Mendukung berbagai tema visual cukup dengan mengganti file CSS.
    - Fully Responsive:Tampilan otomatis menyesuaikan layar Desktop, Tablet, dan Smartphone.

2. # Cara menjalankan
    - Pastikan sudah install node.js
    - buka folder project, lalu buka terminal cmd
    - install lucide-react (npm install lucide-react)
    - jalankan server lokal (npm run dev)
